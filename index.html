Firebase Realtime Sync

Connect a free Firebase project so the whole team sees the same Gantt data. Changes sync every 30 seconds. One-time setup — takes about 5 minutes.

1
Go to console.firebase.google.com → Add project
Name it bubbles-reactor · Disable Google Analytics (not needed) · Click Create project
2
In your project → Realtime Database (left sidebar) → Create Database
Choose any region · Start in test mode (you can lock it down later) → Enable
3
In the database console, copy the URL shown at the top — it looks like:
https://bubbles-reactor-default-rtdb.firebaseio.com
Paste it in the field below.
4
Enter your name so teammates can see who last saved. Share the same Database URL with everyone on the team — that's all they need to connect.
