Firebase Realtime Sync

Connect a free Firebase project so the whole team sees the same Gantt data. Changes sync every 30 seconds. One-time setup — takes about 5 minutes.

1
Go to console.firebase.google.com → Add project
Name it bubbles-reactor · Disable Google Analytics (not needed) · Click Create project
2
In your project → Realtime Database (left sidebar) → Create Database
Choose any region · Start in test mode (you can lock it down later) → Enable
3
In the database console, copy the URL shown at the top — it looks like:
https://bubbles-reactor-default-rtdb.firebaseio.com
Paste it in the field below.
4
Enter your name so teammates can see who last saved. Share the same Database URL with everyone on the team — that's all they need to connect.

Monday Briefing Skill
Run every Monday morning
Analyzes the current Gantt chart and generates a personalized task briefing for every team member. Tells each person exactly what they should focus on this week, based on work stream priorities and current project phase.
📋
Friday Check-In Skill
Run every Friday afternoon
Generates a structured progress check-in script to send to each team member. Asks targeted questions about what was completed, what's still in progress, and any blockers — ready to copy and send via Slack or email.
📊
Weekly Status Report
Generate for stakeholders
Compiles a full project status report — progress by work stream, completion rates per team member, upcoming milestones, and risks. Ready to share with leadership or stakeholders.
🚨
Risk & Blocker Scan
Run anytime
Scans the project for blocked tasks, dependencies at risk, and work streams falling behind. Surfaces items that need immediate attention before they become critical path issues.
